\section{Аннотация}

Данная статья посвящена исследованию объекта Викиданных "Битвы", а также сравнение с объектом "Войны". С помощью SPARQL-запросов, вычисляемых на объектах типа "битва" и "война" в Викиданных, проведено исследование, связанное с географическими координатами, в которм выведен список всех битв, карты с местами, на которых происходили битвы и войны, с заполненными данными в поле "coordinate location" и не заполеннными, списоки битв, у которых есть поле location, но нет поля "coordinate location". Кроме того, сделаны выводы по поводу полноты Викиданных по данной теме.
